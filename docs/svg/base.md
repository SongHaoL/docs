# Svg基础知识

- 2020.05.29

## 定义

> `SVG`全称为`Scalable Vector Graphics`，即`可伸缩矢量图形`。它是一种面向网络的、基于`XML`的标记语言，用来描述二维矢量图形。

## 支持

> `HTML5`文档中可直接内嵌`SVG`图像, 可以忽略文档`DTD`、`SVG版本`、`命名空间`等信息，语法变得很简练。

## 基础形状

SVG为开发者提供了一些预定义的形状元素，以便于快速应用：

- `方形 <rect>`
- `圆形 <circle>`
- `椭圆 <ellipse>`
- `线条 <line>`
- `折线 <polyline>`
- `多边形 <polygon>`
- `路径 <path>`
