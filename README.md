# 关于项目

此项目是使用VuePress搭建的一个博客类型的指南,记录了一些前后端相关的基础知识以及各类在世界项目中遇到的一些问题以及工作的一些总结。欢迎大家关注与star。


# 克隆
> git clone https://github.com/xjl271314/my-blogs.git

# 安装
> npm install  ||  yarn install

# 启动
> npm run dev

# 打包
> npm run docs:build
